# Pseudoclasses
###### [Guia CSS](https://github.com/theleoad/guiaCSS/blob/master/README.md) > [Seletores](https://github.com/theleoad/guiaCSS/blob/master/seletores.md) > Pseudoclasses
---

Uma ṕseudoclasse CSS é uma termo adicionado a um seletor que específica um estado especial desse seletor. Por exemplo, `:hover` pode ser usado para mudar a cor de um botão quando o usuário "coloca o mouse sobre" esse elemento.

```css
button:hover {
  color: blue;
}
```
